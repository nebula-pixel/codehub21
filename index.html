HTML Beginner Guide

HTML stands for HyperText Markup Language. HTML is used to build the structure of websites. Every website uses HTML.

Basic HTML Structure

<!DOCTYPE html>
<html>
<head>
  <title>My Website</title>
</head>

<body>
  <h1>Hello World</h1>
</body>
</html>

The <body> contains everything visible on the website.

Headings

<h1>Big Heading</h1>
<h2>Smaller Heading</h2>

Paragraphs

<p>This is a paragraph.</p>

Links

<a href="https://google.com">Google</a>

Images

<img src="cat.jpg" width="300">
